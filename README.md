# mini-spectrum-maze-reboot

빛의 색에 따라 길이 가시성이 달라지는 3D 미로 퍼즐 게임입니다. Three.js와 순수 JavaScript로 구현했습니다.

## 게임 요약

- 고정 시점에서 미로를 탐색하며 출발점에서 도착점까지 이동
- 현재 선택한 빛의 색과 일치하는 길만 보이는 퍼즐 구조
- 색상 전환과 방향 전환으로 경로를 개척

## 조작 방법

- 방향 회전: WASD 또는 화살표 버튼
- 색 선택: 1 / 2 / 3 키 또는 색상 버튼
- 다시 시작: R

## 진행 목표

- 단일 스테이지 완주
- 색상에 따른 가시성 퍼즐 구현

## 참고

- 고정 시점 퍼즐의 흐름은 Monument Valley의 감각에서 영감을 받았습니다.
